# 🐾 HighQuality Pet Management System

A full-featured **Pet Management Web Application** developed using PHP and MySQL.  
This project allows users to browse, view, and purchase pets, along with authentication and admin management features.

---

## 📌 Features

- 🔐 User Authentication (Login / Register / Logout)
- 🐶 Pet Listing & Detailed View
- 🛒 Pet Purchase System
- 📬 Contact Form
- 🧾 Session & Cookie Management
- 🛠️ Admin Panel for Management
- 📦 Organized MVC-like Structure

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP (Apache)
